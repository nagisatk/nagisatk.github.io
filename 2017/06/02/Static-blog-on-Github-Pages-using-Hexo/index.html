Static blog on Github Pages using Hexo

2017-06-02

Contents

  1. Hexo
  2. 开始使用
  3. 写博客

终于买了域名，开始在Github Pages上写技术博客，虽然说是技术博客，但大概也没什么高深的技术。希望自己可以一直写下去吧。

由于这个静态博客是用Hexo搭建的，所以第一篇就记录一下使用Hexo搭建博客的过程吧。

Hexo

Hexo的自我介绍是“一个快速、简洁且高效的博客框架”。由于对中文的出色支持而在国内拥趸很多，我也是随大流的一员，所以也就用了Hexo。其实Hexo使用起来十分简单，昨天我大部分时间都花在了挑一个喜欢的主题上了，囧rz。

首先，安装一下Node.js和Git，如果是Windows用户，从各自的官网上下载安装就好了。
