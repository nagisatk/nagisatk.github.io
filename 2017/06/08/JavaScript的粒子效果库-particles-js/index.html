基本信息

particles.js是基于HTML5的canvas标签创建粒子效果的轻量级JavaScript库，作者是VincentGarreau，他的Github主页是: https://github.com/VincentGarreau，particles.js的项目主页是：https://github.com/VincentGarreau/particles.js/。

这个方法接受一个HTML元素的id和一个JSON格式的参数，前者是用来生成粒子效果的容器，后者则作为生成的粒子效果的属性。

但这两个参数都可以省略。

如果没有tag_id，会使用默认的id，也就是particles-js；如果没有params，也会有默认的设置效果，如果有params，传入的参数则会重写(overwrite)默认效果。

在项目的Github主页上，作者提供了大部分参数的可选值，根据单词的意思可以知道其作用。

这里列举一部分可选参数：

参数 可选值 效果
particles.number.value number 调整粒子的数量
particles.color.value color 调整粒子的颜色，可以接受多种形式的颜色参数，包括字符串形式的random，或者一个数组
particles.shape.type "circle", "edge", "triangle", "polygon", "star", "image", ["circle", "triangle", "image"] 调整粒子的形状，同样可以接受数组
particles.shape.polygon.nb_slides number 若形状参数中含有polygon，这个参数可以调整多边形的边数
particles.line_linked.enable true/flase 粒子之间是否连线
particles.line_linked.color HEX (string) 上述连线的颜色，似乎只接受#ffffff形式的参数
particles.move.direction "none", "top", "top-right", "right", "bottom-right", "bottom", "bottom-left", "left", "top-left" 粒子移动的方向，默认为none，无规则移动
interactivity.events.onhover.mode "grab", "bubble", "repulse", ["grab", "bubble"] 鼠标悬浮事件属性
interactivity.events.onclick.mode "push", "remove", "bubble", "repulse", ["push", "repulse"] 鼠标单击事件属性
interactivity.events.modes.bubble.size number 若选定bubble事件，此属性可设定气泡大小
