遗传算法

遗传算法（英语：genetic algorithm (GA) ）是计算数学中用于解决最佳化的搜索算法，是进化算法的一种。进化算法最初是借鉴了进化生物学中的一些现象而发展起来的，这些现象包括遗传、突变、自然选择以及杂交等。

模拟遗传

遗传算法中比较重要的两部分，其一是迭代方法，也就是遗传、变异和杂交，其二则是评价函数，也就是适应性。

评价函数

关于适应性的评价，在实际中要解决的问题或许是个非常复杂的问题。但在本例中，因为只是个Demo，所以评价函数也十分简单。

将当前个体的字符串与目标字符串的差异量化成一个整数，这个数值越小则和目标越接近。

假定字符串由字母组成，那么ACE与ABD相比，是后者更接近ABC，这个值则可以由其ASCII码值的方差决定，即：

但是本例中并不是使用的这种方式。

ASCII码值可以看成是由所有ASCII字符组成的字符串的下标，其中有一些是不能打印的字符，所以使用这种方式需要注意这一点。按照这种思路，即使不使用ASCII码值，只要有一个可以进行变异的元素集合就可以了。

所以在Group类中引入了genes成员变量。这个变量出一个字符串，包含能够组成字符串的所有字符，也就是说，即使字符串由汉字组成也是可以的。只要genes变量中有足够的元素就可以。

那么评价函数就不再是ASCII码值的方差，而是当前个体的code与goal的相同下标的字符在genes中下标的方差。

在上例中，如果genes是ABCEFGHIJKD，那么ACE就比ABD更接近目标，因为根据下标方差所计算出的值，前者的eval是2，后者是64。

在本例中，默认的genes是一些ASCII字符。在开始迭代之前，将这些字符进行了随机打乱，这样可以使得文字在跳动的时候显得更“随机”一些。

例如目标字符是L，如果初始字符是A，那么大致的趋势就会从A按照字母序变到L，如果是初始字符是X，则会按照逆字母序变化。

将字符的顺序打乱之后，由于每个字符的前置字符和后置字符是不可预期的，所以变化顺序也就不可预测，看起来更随机一些。（其实没什么用处）
